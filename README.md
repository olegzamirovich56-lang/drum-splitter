# 🥁 Drum Splitter Pro

![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

AI-powered tool for professional drum stem separation. Easily split your drum loops into kick, snare, cymbals, and toms using state-of-the-art neural networks.

Инструмент на базе ИИ для профессионального разделения барабанных партий. Легко разделяйте свои петли на бочку, малый барабан, тарелки и томы с помощью современных нейросетей.

---

## 🇺🇸 English Description

### ✨ Features
* **AI-Engine**: Powered by Larsnet/Demucs for high-precision separation.
* **Drag-and-Drop**: Simple and intuitive interface — just drop your audio file.
* **Auto-Update**: Built-in system to notify you about new versions via GitHub.
* **High Quality**: Supports `.wav`, `.mp3`, and `.flac` formats without quality loss.
