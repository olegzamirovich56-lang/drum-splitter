🥁 Drum Splitter Pro (v1.1.0)
An AI-powered desktop application designed to split drum loops into individual stems: Kick, Snare, Cymbals, and Toms. Built for producers and beatmakers who need high-quality drum separation.

🚀 Features
AI Engine: Powered by Larsnet/DrumSep model.

Drag-and-Drop: Simply drag your audio file into the app.

Batch Info: Tracks processed and failed files in real-time.

Auto-Updates: Notifies you when a new version is available on GitHub.

macOS Optimized: Native menu integration and smooth performance.

🥁 Drum Splitter Pro (v1.1.0)
Десктопное приложение на базе искусственного интеллекта для разделения барабанных петель на отдельные составляющие: Бочка (Kick), Малый барабан (Snare), Тарелки (Cymbals) и Томы (Toms). Создано для продюсеров и битмейкеров, которым нужно качественное разделение ударных.

🚀 Возможности
ИИ Движок: Работает на базе модели Larsnet/DrumSep.

Drag-and-Drop: Просто перетащите аудиофайл в окно приложения.

Статистика: Отслеживание количества обработанных файлов в реальном времени.

Авто-обновления: Приложение само сообщит о выходе новой версии на GitHub.

Оптимизация под macOS: Интеграция в системное меню и нативный вид.
